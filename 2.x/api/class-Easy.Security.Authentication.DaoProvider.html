# saveToCookie( Easy\Security\Authentication\Token\TokenInterface $token, mixed $duration = null )

Saves necessary user data into a cookie. This method is used when automatic login (Easy\Security\Authentication\DaoProvider::$allowAutoLogin) is enabled. This method saves user ID, username, other identity states and a validation key to cookie. These information are used to do authentication next time when user visits the application.
