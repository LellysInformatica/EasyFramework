Class Response

Response is responsible for managing the response text, status and headers of a HTTP response.

By default controllers will use this class to render their response. If you are going to use a custom response class it should subclass this object in order to ensure compatibility.

Direct known subclasses

Easy\Network\BinaryFileResponse, Easy\Network\JsonResponse, Easy\Network\RedirectResponse, Easy\Network\StreamedResponse
